1. Clone the repository
```
$ git clone https://github.com/WiemBEH/LB1_OCR.git
```

2. Check into the cloned repository
```
$ cd LB1_OCR
```

3. UBUNTU : Install the requirements
```
$ bash install_on_ubuntu.sh

4. Start th service 
```
python app.py

```
